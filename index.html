INTRODUCTION

Background of Research on Land Cover Changes After the Semeru Eruption 2021 with SVM and RF Algorithm

Mount Semeru is the highest active volcano on the island of Java

Mount Semeru is located between the administrative areas of Malang Regency and Lumajang Regency with an altitude of 3676 meters above the sea level. Mount Semeru first erupted on November 8, 1818. Then it erupted again on December 4, 2021 with lava floods accompanied by bursts of material and ash rain. The eruption not only threatened the flora and fauna in the Semeru area but also resulted in changes in land cover. Monitoring changes in land cover can be done by processing satellite image data using several methods, one of which is using the supervised classification method with machine learning algorithms. Based on the description above, a study was conducted to compare the capabilities of machine learning in mapping land cover changes using the Google Earth Engine platform with the Random Forest and Support Vector Machine algorithm.

METHODS

Random Forest (RF)

Random Forest developed by Breiman (Breiman, 2001), is one amongst the wide applied powerful ensemble supervised algorithm that uses bootstrap aggregation or “bagging” to create a collection of classifications (using multiple decision trees), with each training tree on a subset of the entire trainig data (Waske & Braun, 2009)

METHODS

Support Machine Learning (SVM)

The support vector machine (SVM) is a type of supervised learning algorithm that is used to solve regression and classification issues. SVM classifiers create an ideal hyperplane in the training stage that separates multiple classes with the fewest misclassified pixels. SVM is used to select the extreme points/vectors that will help create the hyperplane

(Chih-Wei Hsu, Chih-Chung Chang, and Chih-Jen Lin, 2003)
